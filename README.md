# 📌 프로젝트 기능

### 🗓️ 달력

- 증상 기록
- 증상 반복 알림
- 진료 일정 확인

### 🗺️ 지도

- 가까운 병원 및 약국 조회

### 💬 챗봇

- 조건을 적용한 병원 검색
- 병원 예약

### 💊 약봉투

- OCR 기능을 통한 약봉투 인식
- 알러지 약을 기록하여 사전에 예방
